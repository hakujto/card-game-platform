# Card Game Platform

## EN

This repository contains multiple backend implementations of the same card game platform built with different programming languages and frameworks.

Currently implemented:
- Python — Django
- PHP — Symfony
- Ruby — Rails
- TypeScript — Express
- C# — .NET
- Elixir — Phoenix
- Python — FastAPI
- Java — Spring
- PHP — Laravel
- Haskell — Servant
- Clojure — Luminus

More implementations will be added regularly over time.

The goal of this project is to demonstrate:
- adaptability across backend technologies,
- language-agnostic engineering mindset,
- ability to build the same system in different stacks,
- comparison of frameworks and ecosystems.

featuring:
- tournaments,
- peer-to-peer marketplace,
- seasonal system,
- rewards system.
