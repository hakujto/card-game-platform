# CardsProject — Ruby on Rails REST API

## Stack

- Ruby on Rails 7.2 (API mode)
- SQLite3
- RSpec + FactoryBot + Shoulda Matchers
- Rack::CORS

## Quickstart

```bash
cd cards_project
bash setup.sh
bin/rails server
```

API: http://localhost:3000/api/
